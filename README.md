# THE GLOOM

F3 Q Trainer. Spin a random exercise from the F3 Exicon, then learn to lead it.

## Run it

Open `index.html` in a browser. That is the whole app.

## Share it

Host `index.html` on any static host (Netlify, GitHub Pages, Cloudflare Pages)
and send the link. On a phone, use "Add to Home Screen" to install it like an app.

## Develop it

Built as a single self-contained HTML file in vanilla JS, no build step. See
`CLAUDE.md` for project context, constraints, and the roadmap.
